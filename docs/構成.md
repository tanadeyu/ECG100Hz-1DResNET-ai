# ECG 1D-ResNet モデル構成

## モデルアーキテクチャ（軽量版）

```
入力: (batch, 12, 200)
      ↓
┌─────────────────────────────────────────────┐
│  Stem Layer                                 │
│  Conv1d(12→64, k=7, s=2, p=3) + BN + ReLU  │
│  出力: (batch, 64, 100)                     │
└─────────────────────────────────────────────┘
      ↓
┌─────────────────────────────────────────────┐
│  ResNet Layer 1                             │
│  Conv1d(64→64, k=3) + BN + ReLU            │
│  Conv1d(64→64, k=3) + BN                   │
│  + 残差接続 + ReLU                          │
│  出力: (batch, 64, 100)                     │
└─────────────────────────────────────────────┘
      ↓
┌─────────────────────────────────────────────┐
│  ResNet Layer 2                             │
│  Conv1d(64→128, k=3, s=2) + BN + ReLU      │
│  Conv1d(128→128, k=3) + BN                 │
│  + 残差接続（downsample） + ReLU            │
│  出力: (batch, 128, 50)                     │
└─────────────────────────────────────────────┘
      ↓
┌─────────────────────────────────────────────┐
│  Global Pooling + FC                        │
│  AdaptiveAvgPool1d(1) → (batch, 128, 1)     │
│  Flatten → (batch, 128)                     │
│  Linear(128→5) → (batch, 5)                 │
└─────────────────────────────────────────────┘
```

## 設計方針

### モデル軽量化
- **2層構造**: ResNet18（4層）の半分
- **128ch**: ResNet18（512ch）の1/4
- **kernel_size=3**: 計算量削減（1/2.3）
- **maxpool削除**: 信号が短いため不要

### 信号長制限
- **200サンプル（2秒）**: 約2心拍分
- **分類に十分**: P波・QRS複合・T波のパターンが捉えられる

### inplace=Falseの理由
```python
# 残差接続(out += identity)で同一テンソルが複数回使われる場合、
# inplace操作(+=, inplace=True)は逆伝播時にテンソルのバージョン不一致を引き起こす
# RuntimeError: modified by an inplace operation
```
